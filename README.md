# URL Shortener Project

## Technologies
- ExpressJS
- EJS templating engine
- MongoDB Database
- Bootstrap Frontend styling
- Sass : to customize bootstrap


## How to run ?
```bash
npm install

npm run start

// If writing in main.scss, run in another terminal
npm run sass

```
Now website will be visible at - http://localhost:300
